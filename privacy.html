Privacy Policy

MAWed – Browser Extension

This extension does not collect, store, or transmit any personal data.

All data related to watched videos is saved locally in your browser using localStorage. Nothing is sent to any remote servers.

The extension does not use any cookies, tracking tools, or analytics.

Permissions

This extension requests permission to access https://www.youtube.com/* only to modify the interface and allow you to manually mark videos as watched. No data is shared externally.

No Data Collection

We do not collect:

  • Personally identifiable information
  • Location or browsing history
  • Passwords or financial information
  • User communications or content

Contact

If you have questions, please contact the developer through the Chrome Web Store listing.
